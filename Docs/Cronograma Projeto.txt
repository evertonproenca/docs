CRONOGRAMA DO PROJETO

VIEW

1ª SEMANA: CONSTRUIR O FORMULÁRIO
- CRIAR OS CAMPOS DADOS CADASTRAIS;
- CRIAR OS CAMPOS MARKETING/COMUNICAÇÃO;
- CRIAR OS CAMPOS PREÇOS;
- IMPLEMENTAR A VIEW NO PHP;

2ª SEMANA: CONTINUAR A CONSTRUÇÃO DO FORMULÁRIO
- CRIAR OS CAMPOS VENDAS;
- CRIAR OS CAMPOS COMPRAS;
- CRIAR OS CAMPOS CLIENTES;
- IMPLEMENTAR A VIEW NO PHP;

3ª SEMANA: FINALIZA A CONSTRUÇÃO DO FORMULÁRIO E IMPLEMENTAR O BOOSTRAP
- CRIAR OS CAMPOS RECURSOS HUMANOS;
- CRIAR OS CAMPOS ADMINISTRATIVO/FINANCEIRO/CONTÁBIL
- INSERIR O TWITTER BOOSTRAP NO FORMULÁRIO;

4ª SEMANA:  FINALIZAR A PARTE DA VIEW;
- FINALIZAR A IMPLEMENTAÇÃO DA VIEW;

5ª SEMANA: FINALIZAR
- FINALIZAR FORMULÁRIO - COMPLETO

CONTROLLER

1ª SEMANA: INICIAR A LIGAÇÀO DO BANCO DE DADOS E DA VIEW
- IMPLEMENTAR OS COMANDOS INSERT, DELETE E SELECT DOS CAMPOS: DADOS CADASTRAIS, MARKETING E COMUNICAÇÃO, PREÇOS;
- INTERLIGAR A PRIMEIRA PARTE DA VIEW COM A PRIMEIRA PARTE DO MODEL;

2ª SEMANA: DAR CONTINUIDADE NA LIGAÇÀO DO BANCO DE DADOS E DA VIEW
- IMPLEMENTAR OS COMANDOS INSERT, DELETE E SELECT DOS CAMPOS: VENDAS, COMPRAS E CLIENTES;
- INTERLIGAR A SEGUNDA PARTE DA VIEW COM A SEGUNDA PARTE DO MODEL;

3ª SEMANA: FINALIZAR A LIGAÇÀO DO BANCO DE DADOS E DA VIEW
- IMPLEMENTAR OS COMANDOS INSERT, DELETE E SELECT DOS CAMPOS: RECURSOS HUMANOS E ADMINISTRATIVO/FINANCEIRO/CONTÁBIL
- INTERLIGAR A TERCEIRA PARTE DA VIEW COM A TERCEIRA PARTE DO MODEL;

4ª SEMANA:  FINALIZAR A PARTE DA VIEW;
- REALIZAR TESTES FINAIS DO PROJETO;

5ª SEMANA: FINALIZAR
- FINALIZAR FORMULÁRIO - COMPLETO

GERENTE

1ª SEMANA: INICIANDO A GERENCIA DO PROJETO
- GERAR O CRONOGRAMA;
- DISTRIBUIR AS TAREFAS DA PRIMEIRA SEMANA;
- REALIZAR O RELATÓRIO DA SEMANA;
- VER O ANDAMENTO DO PROJETO NA SEMANA INICIAL;

2ª SEMANA: ANALISAR O DESENVOLVIMENTO DO PROJETO
- DISTRIBUIR AS TAREFAS DA SEGUNDA SEMANA;
- REALIZAR O RELATÓRIO DA SEMANA;
- VERIFICAR O ATUAL DESEMPENHO DO PROJETO;
- LISTAR OS PRINCIPAIS ERROS ENCONTRADOS NO PROJETO;
- ANALISAR POSSIVEIS SOLUÇÕES PARA CORREÇÃO DE ALGUNS ERROS;

3ª SEMANA: GERENCIAR FASES DE TESTES
- DISTRIBUIR AS TAREFAS DA TERCEIRA SEMANA;
- REALIZAR O RELATÓRIO DA SEMANA;
- VERIFICAR O ATUAL DESEMPENHO DO PROJETO;
- VERIFICAR SE OS ERROS ENCONTRADOS NA SEMANA ANTERIOR FORAM CORRIGIDOS;

4ª SEMANA:  ACOMPANHAR A FINALIZAÇÃO DO PROJETO;
- DISTRIBUIR AS TAREFAS DA QUARTA SEMANA;
- REALIZAR O RELATÓRIO DA SEMANA;
- VERIFICAR O DESEMPENHO FINAL DO PROJETO;
- REALIZAR UM RELATÓRIO FINAL DO PROJETO;

5ª SEMANA: FINALIZAR
- DISTRIBUIR AS TAREFAS DA QUINTA SEMANA;
- REALIZAR O RELATÓRIO DA SEMANA;
